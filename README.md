## Requirements

- iOS 16.0+
- Xcode 15.0+
- Swift 5.9+

## Installation

1. Clone the repository
2. Open `RecipeBook.xcodeproj` in Xcode
3. Build and run the project

## Technologies

- SwiftUI for modern UI
- Core Data for persistence
- Swift Concurrency
- Native iOS frameworks
